Minha formação profissional






O primeiro passo para desenvolver um site é organizar e estruturar todos os elementos que estarão presentes em cada página. Para isso isso, a linguagem HTML é utilizada. Esta foi a primeira linguagem que eu dominei.







Depois de estruturado, o site precissa ser estilizado. É nessa parte que entra o CSS. Inclusive, essa foi a linguagem que mais me fez sofrer até então. Perdi muitas horas de sono aprendendo a usá-la e entendendo a lógica de seu funcionamento. Contudo, após tanto sofrimento, consegui dominá-la também.





Depois de estruturar e estilizar um site, é preciso criar o script que rodará por trás de tudo. Existem diversas linguagens de programação que podem ser usadas, mas a única que eu domino até o momento é o javascript. Sinceramente, eu imaginei que esta seria a parte mais complicada na criação de um site, porém rapidamente consegui me adaptar ao javascript.






C foi a primeira linguagem de programação que eu aprendi. Apesar de não ser utilizada na criação de sites e de ser uma linguagem antiga, é possível criar diversos programas muito uteis com ela. Além disso, saber usar C fez com que eu tivesse maior facilidade na hora de aprender javascript.
